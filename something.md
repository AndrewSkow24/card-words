## Vite

Vite - удобный инструмент, который может собрать базовый React-проект.

Не нужно вручную прописывать настройки.

. Точка в конце позволит развернуть проект прямо в указанной попке

**index.html** - главный html - файл приложения, здесь подключается React

**src** - папка с исходным кодом

**main.jsx** - точка входа в приложение, где React соединён с DOM

**App.jsx** - главный компонент приложения

**assets** - папка для изображений, шрифтов и других статических файлов
